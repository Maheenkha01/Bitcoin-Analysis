# Bitcoin-Analysis

This quantitative study investigates how Bitcoin's proof-of-work difficulty retargeting mechanism responds to extreme hashrate disruptions across a five-year window (2020–2025). Abnormal difficulty events were defined as adjustments exceeding 2σ (±27.6%), identifying two major shocks tied to China's mining crackdown in 2021: a +21.35% spike at Block 683424 and a -27.63% collapse at Block 689472. Structural breaks were validated using Chow tests across four events — the 3rd and 4th halvings, and both phases of the China Ban — all returning highly significant F-statistics. OLS regression confirmed that shock magnitude has no meaningful effect on block time variance (R² ≈ 0.01–0.02), while an ARIMA(1,0,0) model showed shocks carry no autocorrelation. However, a GARCH(1,1) model revealed that volatility itself is highly persistent (α + β = 0.96), indicating disruptions cluster into regimes rather than resolve instantaneously. Recovery analysis showed block times returning to target within 8–12 hours across all events, without breaching the protocol's 4x adjustment cap.
